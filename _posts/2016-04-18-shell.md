<h5 id="ls"> ls 命令 </h5>
```
  ls 显示目录和文件
  参数：
    -l 显示长列表
    -h 显示常用单位
    -a 显示影藏文件
    -i 显示文件inode number
    -t 按修改时间排序
    -S 按文件大小排序
    -r 取反（大小，时间排序）
```

<h5 id="ps"> ps 命令 </h5>
```
  ps 查看进程信息
  参数：
    -A 显示所有进程
    -a 显示当前终端的进程
    -u 显示进程所属的用户
    -x 
    -o pid,command   指定输出的格式
```
